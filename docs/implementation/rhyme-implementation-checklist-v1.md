# LyricLytic 韻ガイド実装チェックリスト v1

## 1. 目的

本書は、`docs/requirements/rhyme-analysis-v1.md` を実装へ落とすための運用チェックリストである。  
韻ガイドは `辞書層 / 正規化層 / 候補抽出層 / LLM補正層 / UI層` の責務が分かれており、場当たり的に足すと破綻しやすい。  
そのため、段階的に実装を進めつつ、各段階で確認すべき観点を明示する。

## 2. 運用ルール

- 各項目は `未着手 / 進行中 / 完了 / 保留` のいずれかで管理する
- 実装を入れたら `review-log.md` に Gate を追記する
- build / check / test の確認結果をその都度残す
- UI だけ先行変更して backend 契約を増やしすぎない
- LLM は `韻候補探索の主処理` に使わない

## 3. 現在地

- 状態: `進行中`
- 現在の実装:
  - フロントに韻ガイド表示あり
  - フォールバックのローマ字 / 母音 / 子音変換あり
  - Tauri 実行時は `SudachiPy + SudachiDict-core` 解析あり
  - failure 時はフォールバックへ戻る
- まだ無いもの:
  - NEologd 補完
  - UniDic 検証系
  - 韻候補抽出 UI
  - 再ランキング / 近似候補 UI
