# v1.2.0 Release Runbook

## Automated Gate

Run from a clean dependency state:

1. `npm ci`
2. `npm run lint && npm run build`
3. `npm run test:run -- --maxWorkers=1 && npm run test:coverage`
4. `npm run test:e2e -- --workers=2`
5. `npm run audit:prod && npm run audit:all`
6. `cd src-tauri && cargo fmt -- --check`
7. `cargo clippy --all-targets --all-features -- -D warnings`
8. `cargo test --all-features && cargo audit --json`
9. `npm run tauri:build`

Required results are: frontend coverage each metric >=80%, production npm moderate+ = 0, all npm high+ = 0, Rust vulnerabilities = 0, Clippy warnings = 0, and all tests pass.

## CI Gate

The Windows/macOS matrix must pass npm/Rust checks, audits, Playwright, and platform bundle generation. It uploads cargo-audit JSON, failed Playwright diagnostics, and successful bundles. Windows requires MSI/NSIS; macOS requires `.app`/`.dmg`; each job verifies the bundled Sudachi dictionary.

## Windows Formal Gate

Follow [Windows offline QA](windows-offline-qa.md) in a disposable Windows environment. The required evidence location and redaction policy are in [evidence](v1.2.0/evidence/README.md). Do not release until all P0/P1 cases pass.
