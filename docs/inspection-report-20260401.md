## 修正完了した不具合

### ✅ BUG-001: 同名セクションの連番付与
- **修正内容**: `sectionUtils.ts` に `generateUniqueSectionName` 関数を追加
- **詳細**: 同名セクション追加時に "Verse", "Verse 2", "Verse 3" のように連番を付与

### ✅ BUG-002: インポート機能の Tauri 対応
- **状態**: 既に修正済み
- **詳細**: `@tauri-apps/plugin-fs` の `readFile` を使用、複数エンコーディング対応済み

### ✅ BUG-003: .lyrlytic.zip エクスポート機能
- **状態**: 既に修正済み
- **詳細**: Tauri dialog/fs plugin を使用した正式なエクスポート機能実装済み

### ✅ BUG-004: タグ検索機能
- **状態**: 既に修正済み
- **詳細**: SearchPanel にタグ検索タブ追加、searchUtils でタグ検索ロジック実装済み

### ✅ BUG-005: コピー設定の永続化
- **修正内容**: `CopyOptionsPanel.tsx` で localStorage への永続化を実装
- **詳細**: `loadOptions` / `saveOptions` 関数を追加し、設定変更時に自動保存

### ✅ BUG-006: StyleProfile の論理削除・復元
- **状態**: 既に修正済み
- **詳細**: trash モジュールで統一的な削除・復元機能実装済み

### ✅ BUG-007: revision_notes の型不一致
- **修正内容**: `models.rs` と `types.ts` で `version_section_id` を必須に変更
- **詳細**: スキーマの NOT NULL 制約に合わせて型を統一

### ✅ BUG-008: 未使用インポート
- **状態**: 既に解消済み
- **詳細**: リファクタリング時に削除済み、ビルド警告なし
