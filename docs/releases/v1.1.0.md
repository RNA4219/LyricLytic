# LyricLytic v1.1.0

編集体験と LLM 補助を大きく強化したマイナーアップデートです。

## ハイライト

- セクション並び替えを `React DnD` ベースへ刷新
- ドラッグ中の追従プレビューとカード操作感を改善
- `Lyrics / Style / Vocal` の生成補助プロンプトを見直し
- Suno 向け公開プロンプト辞書をローカル同梱し、生成時に参照
- Suno 辞書を shard 分割し、必要時だけ読み込む構成へ最適化
- 韻ガイドの Python 実行フォールバックを改善

## UI と使い勝手

- AI 補助パネル内のモードタブを中央配置に調整
- 感情 / トーンのセレクト UI を周辺デザインに馴染む見た目へ改善
- セクションカードのドラッグ中に浮遊プレビューを表示
- セクション並び替え時の視認性と追従感を改善

## AI / LLM

- `続きを書いて` と `言い換え` の動作を見直し
- 空入力の `言い換え` を送信しないよう修正
- 感情 / トーン指示をより強くプロンプトへ反映
- `Style / Vocal` 生成時に Suno 辞書の関連候補を自動注入
- Suno 辞書更新用の週次 GitHub Actions ワークフローを追加

## パフォーマンス

- Suno 辞書を用途別 bucket に分割
- 起動時には重い辞書本体を読まず、`Style / Vocal` 利用時だけ遅延ロード
- `Vocal` 系は特に軽い chunk 構成へ最適化

## 修正

- Windows 環境で `python` が異常終了した際も `py -3` にフォールバックするよう修正
- 韻ガイドが不要に `Fallback` 表示へ落ちるケースを改善

## 補足

- 現在の生成品質は接続しているローカルモデルの特性に依存します
- フロント側の分割ロードと生成導線は実機確認済みです
