# LyricLytic v0.1.0

初回の実用版リリースです。

## ハイライト

- `llama.cpp` を LyricLytic から直接起動
- 歌詞 / Style / Vocal を 1 つの画面で管理
- 韻ガイドでローマ字 / 母音 / 子音を可視化
- スナップショット保存とバージョン差分比較
- 削除済みアイテムの復元と完全削除

## UI と使い勝手

- ホーム画面を整理し、新規プロジェクトと既存カード中心の構成に調整
- 実機スクリーンショットを README に追加
- スクロールバーやダイアログ、各種パネルの見た目を調整
- 保存や非表示操作にトースト表示を追加

## AI / LLM

- `llama.cpp` 一本化
- GGUF ファイル指定での運用を前提化
- `Lyrics / Style / Vocal` 生成補助を改善
- AIモデル頻出表現チェックを追加

## 編集機能

- `ALL` 編集とセクション編集の挙動を改善
- BPM の保存と復元をスナップショット / 自動保存に対応
- 行ごとの韻ガイド表示と文字数表示を改善
- セクションの並べ替え UI を改善

## ドキュメント

- 日本語 README を一般向けに整理
- 英語版 README を追加
- 主要ソフトウェアと辞書のライセンス一覧を追加

## 補足

- Windows が主対応です
- macOS は起動するように実装していますが、実機未検証です
- サポート予定期間は 2026/07/01 までです
